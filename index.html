Where to next with research?

Re-design QWT as an ethical open-source research tool

  • Share code and produce documentation

  • Introduce new features for data collection, including tracking changes to submissions made by users, and storing the data generated by users privately and in de-identifiable format

Leverage large-scale social learning environment for peer assessment

  • Incorporate pair-wise comparative judgements

  • Rank writing using Bradley-Terry modelling approaches

Use data collected, with appropriate ethics approval:

  • Explore learner engagement and submissions over time

  • Investigate AI integrations for supporting feedback systems
